# MinimalInventoryItems

Opt-in sample loaded **only via McMaster** (`plugin.json` + DLL). Enable `Plugins.Enabled` in `config/server.json`.

```bash
dotnet build plugins/MinimalInventoryItems/MinimalInventoryItems.csproj
```

Registers cobblestone / wooden sword / stick in Construction / Equipment / Items.

See [plugins architecture](../../docs/pt_br/plugins/README.md).
