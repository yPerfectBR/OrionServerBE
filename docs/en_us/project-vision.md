# Project Vision

OrionServer exists to provide a custom Bedrock server core for builders who want control over exactly what runs in production.

## Core Idea

The server should not include features that are not needed by the project using it. OrionServer intentionally focuses on minimal mandatory systems and keeps the rest opt-in.

## What OrionServer Prioritizes

- A lightweight core runtime
- Strong extension surface for plugins and custom systems
- Performance and scaling for heavy shared-world scenarios
- Infrastructure that can leverage modern multi-core CPUs

## Typical Use Cases

- Minigame networks with custom game logic
- Fixed-world projects (RPG, event maps, apocalypse scenarios)
- Specialized gameplay servers where vanilla parity is not the main goal
