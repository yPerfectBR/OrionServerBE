# Issue Guidelines

Please open separate issues for separate problems.

## Report Structure

For each issue, include:

- what happened
- when and where it happened
- exact reproduction steps
- expected behavior vs actual behavior

## Useful Extras

- For in-game issues: include client version and world/server configuration.
- A short public or unlisted video showing the problem can significantly speed up diagnosis.

Detailed, reproducible issues help keep turnaround time low.
